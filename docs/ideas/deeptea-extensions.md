# DeepTea 2.x — Social & Smart Extensions

Этот документ описывает надстройки над базовым ядром DeepTea (BLE + P2P + офлайн-ваучеры):

1. Social Tip Jar — "Чаевые с историей"
   - Социальный слой поверх транзакций.
   - Сообщения благодарности, медиа, теги.
   - Будущий NFT-слой — асинхронно и опционально.

2. Tip Battles — "Соревновательные чаевые"
   - Временной "баттл" между участниками по сумме чаевых.
   - Лидерборды, ачивки, игровые сценарии.

3. Merchant Dashboard — "Аналитика для бизнеса"
   - Панель для заведений.
   - Реальное время + агрегированная аналитика.

4. Charity Tips — "Чаевые с миссией"
   - Сплит чаевых: часть получателю, часть благотворительности.
   - Прозрачные отчёты, далее — возможно on-chain слой.

5. AR Tips — "Дополненная реальность чаевых"
   - AR-сессии, визуал передачи чаевых.
   - На данном этапе — backend-скелет + описание.

6. Predictive / AI Tips — "AI-предсказания чаевых"
   - Рекомендации сумм.
   - Сейчас — rule-based движок, позже — модель.

MVP-фокус текущего этапа:
- Полноценный Social Tip Jar.
- Лёгкий Merchant Dashboard.
- Простой Tip Battles (без тяжёлой игровой логики).
- Базовый Charity split.
- AR / AI — как скелеты модулей и документация.
