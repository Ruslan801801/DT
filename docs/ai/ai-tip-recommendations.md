# AI Tip Recommendations — MVP

Цель:
- не строить сразу тяжёлую ML-платформу,
- дать "умный" выбор предустановленных сумм на основе:
  - истории чаевых пользователя,
  - суммы чека (если есть),
  - времени суток,
  - базовой оценки качества сервиса.

MVP:
- rule-based движок `AiTipService`.
- API: `/api/ai/recommend-tip` возвращает:
  - recommended_amount
  - alternatives[] (щедро/стандарт/экономно/процент от чека)
  - explanation (текст).
