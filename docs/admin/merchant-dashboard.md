# Merchant Dashboard (MVP)

Цели:
- Дать заведению простой обзор чаевых за сегодня.
- Показать последние транзакции.
- Подготовить основу для более сложной аналитики.

Состав:
- backend:
  - MerchantAnalyticsService — агрегирует данные по транзакциям.
  - MerchantGateway — WebSocket-обновления.
- web-admin:
  - Отдельный SPA/страница, которая подключается к `wss://api/merchant?merchantId=...`
  - Рендер карточек: "Чаевые сегодня", "Средний размер" и список последних чаевых.
