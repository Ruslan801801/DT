# AR Tips — концепт и текущий статус

На данном этапе:
- AR Tips НЕ являются частью MVP.
- В коде представляется только backend-скелет `ARSessionService`.
- Мобильная часть будет реализована позднее на базе ARKit / ARCore / ViroReact или альтернатив.

Основные идеи:
- AR-сессия связывает двух участников.
- Сессия хранит метаданные: расстояние, длительность, эффекты.
- Транзакция P2P связывается с AR-сессией.
