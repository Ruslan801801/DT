# DeepTea Monorepo

Стартовый каркас для P2P чаевых по BLE (онлайн + офлайн ваучеры).

## Структура
- `mobile/` — React Native (Expo)
- `native/` — нативные BLE модули (iOS/Android)
- `server/` — NestJS API
- `infra/` — docker-compose, мониторинг
- `shared/` — общий код (типы/утилиты)
- `docs/` — документация (OpenAPI, runbooks, legal)
